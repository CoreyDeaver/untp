# A command line tool to split TexturePacker publish file.

## Install

### 1. use pip

`pip install untp`

### 2. from source

`git clone git@github.com:justbilt/untp.git`

`python setup.py install`

## How to use ?

`untp your_plist_file.plist`

or

`untp your_plist_file.plist your_image_file.png`
